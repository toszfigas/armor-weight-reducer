# Armor Weight Reducer

Automatically halves the weight of equipped armor in D&D 5e for Foundry VTT.

## Installation

Use this manifest URL in Foundry:
```
https://github.com/toszfigas/armor-weight-reducer/releases/latest/download/module.json
```

## Usage

Simply equip armor and the weight will be automatically changed. Unequip to restore original weight.
